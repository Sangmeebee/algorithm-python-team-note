## 내장함수

- 내장함수
  - sum()
    - sum([1, 2, 3, 4, 5])
  - min()
  - max()
  - eval() 
    - result = eval("(3+5)*7") : 문자열을 그대로 계산
  - sorted()
    - sorted([9, 1, 8, 5, 4])
    - sorted([9, 1, 8, 5, 4], reverse=True)
    - sorted(array, key=lambda x:x[1], reverse=True)      //array=[('김상민', 1), ('방유빈', 2)]
